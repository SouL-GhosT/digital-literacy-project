# Task 3 – Coding & Collaboration Platforms

## Part A – Coding Practice (HackerRank)

**Platform used:** [HackerRank](hackerrank.com)

**Challenge completed:** Java 1D Array (Java)

**What the challenge involved:**
The 1D Array Java challenge on HackerRank is a beginner-level problem that involves working with arrays. The task required:

1. Reading an integer `n` from stdin denoting the number of integers to follow
2. Reading `n` integers from stdin and storing each in a variable
3. Creating an array `arr` capable of holding `n` integers
4. Modifying the input loop to save each sequential value to its corresponding location in the array (index 0, 1, 2, etc.)
5. Printing each element of the array on a new line

This challenge helps beginners understand array declaration, initialization, loops, and how to iterate through arrays to store and retrieve values.

**Steps I followed:**
1. Created an account on HackerRank using my college email
2. Navigated to Practice → Data Structures → Arrays
3. Opened the "1D Array" challenge
4. Read the problem statement and understood the requirements for array creation and manipulation
5. Declared an integer array `arr` of size `n`
6. Modified the input loop to store each value at index `i` using `arr[i] = scan.nextInt()`
7. Created a loop to print each element of the array
8. Clicked "Run Code" to test, then "Submit" to complete
9. Received a score and completion badge

## Reflection 

For Task 3, I used two platforms — HackerRank for coding practice and Google Forms
for building a collaborative quiz.

On HackerRank, I completed the "1D Array" beginner challenge in Java. This problem
helped me understand fundamental data structure concepts like array declaration,
initialization, and iteration. Working with arrays is essential in programming,
and tackling this problem improved my understanding of how to store and manipulate
multiple values using a single variable. The experience of navigating the platform,
understanding the problem requirements, and implementing the solution was very useful.
HackerRank is widely used by companies for technical interviews, so getting familiar
with it early is a great advantage.

For the Google Forms part, I created a five-question Digital Literacy Awareness Quiz
that could genuinely be used to test my batchmates' knowledge of online safety,
passwords, and digital tools. I included both multiple choice and short answer
questions to make it engaging. The linked Google Sheet automatically records all
responses, which makes it easy to analyse results.

Both tools will be very useful academically — HackerRank for practicing coding before
placements, and Google Forms for collecting data for research projects, feedback
surveys, and group assignments throughout my B.Tech.
